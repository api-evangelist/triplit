# Triplit (triplit)

Triplit is an open-source full-stack database that syncs data between server and browser in real-time, combining a client-side store, server sync engine, and schema-defined collections. Built for TypeScript developers, it provides real-time sync with CRDT-based conflict resolution, local caching, offline support, optimistic updates, and relational querying — making it suitable for collaborative and local-first web applications. Triplit is self-hosted and works with React, Svelte, Vue, Solid, Angular, and React Native.

## APIs

- **Triplit HTTP API** — RESTful API for fetch, insert, bulk-insert, update, delete, and healthcheck operations. JWT Bearer auth. [Docs](https://www.triplit.dev/docs/http-api)
- **Triplit Client SDK** — TypeScript/JavaScript SDK with real-time subscriptions, offline support, and framework bindings. [Docs](https://www.triplit.dev/docs/client)
- **Triplit CLI** — CLI for scaffolding projects, running local dev servers, and managing schema migrations. [Docs](https://www.triplit.dev/docs/cli)
